# EVA-2 — Обзор проекта (для менеджеров)

Документ описывает **что сделано в демо** и **как это работает** простыми словами.
Без кода — только логика и список готовых экранов.

---

## 1. Что это за продукт

EVA-2 — это **витрина недвижимости для застройщика**: сайт, на котором покупатель
выбирает квартиру (а также паркинг, кладовку или коммерческое помещение) в жилых
комплексах застройщика.

Ключевые особенности демо:

- **Несколько застройщиков на одной платформе.** Каждый застройщик получает свой
  раздел сайта со своим логотипом, цветами и телефоном. В демо настроено два:
  «Атлас» и «Самолёт».
- **AI-помощник.** Встроенный чат-консультант, который подбирает квартиры и отвечает
  на вопросы о комплексах.
- **Админ-панель.** Простой раздел для управления застройщиками (название, телефон,
  логотип, фирменные цвета).

## 2. Главная логика: бэкенд работает «через фид»

Это самая важная часть для понимания, как устроена система.

### Что такое «фид»

**Фид** — это единый поток данных обо всех объектах недвижимости: список жилых
комплексов, корпусов, всех квартир и помещений, их цены, площади, статусы (в продаже /
забронировано), а также планировки.

В реальной эксплуатации такой фид застройщик отдаёт **выгрузкой из своей CRM или базы
данных** (как правило, по расписанию). Сайт не хранит данные о квартирах сам — он
**берёт их из фида** и показывает покупателю.

### Как это работает в демо

```
   Фид (источник данных о квартирах)
            │
            ▼
   Бэкенд EVA-2  ──►  превращает данные фида в удобный вид
            │          (комплексы, корпуса, квартиры, планировки)
            ▼
   Сайт (фронтенд)  ──►  показывает покупателю
```

- Сейчас вместо реальной выгрузки застройщика используется **демо-фид** —
  подготовленный набор тестовых данных (4 комплекса, корпуса и ~155 помещений).
- **Весь сайт уже работает поверх этого фида.** Списки комплексов, карточки квартир,
  фильтры, планировки, AI-подбор — всё берёт данные из одного источника.
- Благодаря этому переход на реальные данные застройщика — это **замена источника
  фида**, без переделки сайта. Логика отображения остаётся той же.

### Что это даёт бизнесу

- **Быстрое подключение нового застройщика** — достаточно подать его фид.
- **Данные всегда из одного места** — нет ручного дублирования цен и наличия.
- **Сайт не зависит от формата CRM застройщика** — бэкенд приводит данные к единому виду.

## 3. AI-помощник

В правой части сайта доступен чат с AI-консультантом.

- Помогает **подобрать квартиру** по запросу обычными словами («двушка до 12 млн с
  видом на парк»).
- **Берёт данные только из фида** — не выдумывает цены, площади и наличие.
- Понимает контекст страницы: если покупатель открыл конкретную квартиру и спрашивает
  «а сколько стоит эта?», AI понимает, о какой квартире речь.
- Найденные варианты показывает **карточками** прямо в чате, а не длинным текстом.
- История переписки сохраняется для покупателя (привязана к его сессии в браузере).

## 4. Готовые страницы сайта (фронтенд)

Все страницы работают внутри раздела конкретного застройщика (например, `/atlas/...`
или `/samolet/...`) и автоматически окрашены в его фирменные цвета.

### Покупательская часть

- **Список жилых комплексов** — стартовая страница. Показывает все ЖК застройщика
  с ценами «от» и количеством квартир. Можно переключиться в **режим карты** и
  посмотреть расположение комплексов.
- **Страница комплекса (лендинг ЖК)** — подробная карточка одного комплекса:
  описание, статистика по типам квартир (студии, 1-, 2-, 3-комнатные и т.д.),
  расположение на карте, переход к выбору квартир.
- **Каталог квартир** — список всех помещений выбранного комплекса с фильтрами
  (комнаты, цена, площадь, этаж, отделка, акции и др.). Доступны **три режима
  просмотра**:
  - список карточек,
  - **«шахматка»** (сетка квартир по этажам и секциям),
  - **визуальный выбор по планам** (выбор корпуса → этажа → квартиры прямо на
    изображении планировки).
- **Страница квартиры** — детальная карточка: галерея, характеристики, цена,
  условия покупки (ипотека, рассрочка), кнопка обратного звонка, добавление в
  избранное, а также подбор **«квартир с такой же планировкой»**.
- **Просмотр корпуса** — интерактивный план корпуса: выбор этажа на изображении.
- **Планировка этажа** — план этажа с кликабельными квартирами.
- **Избранное** — сохранённые покупателем квартиры.

### Сквозные элементы

- **AI-чат** — доступен поверх страниц.
- **Форма обратного звонка** — заявка на связь с менеджером.
- **Адаптивный интерфейс** в фирменном стиле застройщика.

### Служебная часть

- **Админ-панель** (`/admin`) — вход по паролю. Список застройщиков и редактирование
  их данных (название, телефон, логотип, фирменные цвета) с живым предпросмотром темы.
- **UI-Kit** (`/ui-kit`) — внутренняя страница с библиотекой компонентов интерфейса
  (для разработки и дизайна, не для покупателей).

## 5. Итог: что готово в демо

- ✅ Витрина недвижимости, полностью работающая **на основе фида**.
- ✅ Поддержка **нескольких застройщиков** с индивидуальным брендингом.
- ✅ Полный путь покупателя: список ЖК → комплекс → каталог → квартира → заявка.
- ✅ Три режима выбора квартир, включая визуальный выбор по планировкам.
- ✅ **AI-помощник** для подбора квартир по данным фида.
- ✅ **Админ-панель** для управления застройщиками.

**Следующий шаг для продакшена:** подключить реальный фид застройщика вместо
демо-данных — сайт к этому уже готов.
